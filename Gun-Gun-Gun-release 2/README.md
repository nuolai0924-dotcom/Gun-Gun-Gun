# 数字枪火

一个用 HTML、CSS 和 JavaScript 写的浏览器小游戏。玩法结合跑酷选门、四则运算、答题补弹和 Boss 瞄准射击。

## 玩法简介

玩家在跑道上左右移动，选择不同的数学门来改变最终子弹数。进入 Boss 战后，最终数字会四舍五入成为子弹数量，最多 50 发。Boss 战中需要一边瞄准移动的怪物射击，一边回答数学题来补充子弹。

## 功能

- 跑酷阶段包含加法、减法、乘法、除法门
- 开局获得 100 金币，可以先在大厅升级
- 战斗中不能升级，只有回到大厅才能使用金币升级
- 支持小数数字，最终子弹数会四舍五入
- Boss 战中点击或触摸屏幕瞄准射击
- 子弹伤害会随着伤害升级提高
- Boss 血量会随着关卡提高
- Boss 血量低于一半时会嘶吼并加速
- Boss 每 40 秒释放一次专属技能，需要移动躲避
- 金刚技能：猛砸
- 哥斯拉技能：甩尾
- 机械哥斯拉技能：原子吐息
- 被 Boss 技能命中会减少 5 点生命值
- 答对题目获得 10 发子弹，答错扣 10 点生命值
- 子弹达到 50 发后，题目升级为解方程
- 子弹满 50 发时 Boss 回复 100 血，溢出血量会增加最大血量
- 随机城市 Boss 战场景
- 枪、怪物和城市背景使用 Canvas 绘制
- 包含射击、四则运算门、胜利和失败语音音效

## 操作方式

- 大厅：先使用金币升级，再点击开始按钮进入下一关，不能操控炮台
- 跑酷阶段：使用 `A / D`、方向键，或拖动/触摸来左右移动
- Boss 战：点击或触摸怪物位置进行射击
- Boss 技能阶段：使用 `W / A / S / D`、方向键，或拖动/触摸来躲避
- 答题：在底部输入框输入答案并提交
- 暂停：点击右上角暂停按钮，或按空格键

## 运行方式

直接用浏览器打开：

```text
index.html
```

也可以用本地静态服务器运行，例如：

```bash
python3 -m http.server 5173
```

然后访问：

```text
http://127.0.0.1:5173
```

## GitHub Pages 部署

1. 把项目上传到 GitHub 仓库
2. 进入仓库的 `Settings`
3. 打开 `Pages`
4. Source 选择 `Deploy from a branch`
5. Branch 选择 `main`，目录选择 `/root`
6. 保存后等待 GitHub 生成访问链接

## 技术说明

这个项目不依赖任何外部库，所有游戏逻辑、画面、音效和 UI 都在 `index.html` 中完成。音效使用浏览器内置 Web Audio API 和语音合成 API 生成。
